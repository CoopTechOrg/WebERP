# 概要
案件管理 WebERP作るよ

# 主な機能
## 受注管理
- いつ、誰から、案件を受けたか。どんな内容か。
- 案件の内容
- 納期
- 請求先、請求金額

## 売上管理
- 請求したものが入金されたか
- 誰あての請求が売り上げられたか
- 売上管理と入金管理は別？

## 仕入管理
- 経費で何か仕入れたらその電子領収書管理

## 工数、勤怠管理
- いつ、どのくらい時間かけて何の作業をしたか
  - 自分の力量が計れそう
